1. ✅ 패키지 설치: `./start.sh` 실행

## 💡 주요 특징

### 1. 매너 점수 시스템 (핵심!) ⭐
```
사용자 가입 → 초기 점수 50%
     ↓
모임 참여 후 평가
     ↓
칭찬 + 후기 + 매너 평가
     ↓
점수 자동 계산 (0~100%)
     ↓
프로필에 하트로 표시 ❤️
```

### 2. 실시간 기능
- Socket.io로 실시간 인원수 업데이트
- FCM 푸시 알림 준비
- Redis 캐싱

### 3. 보안
- JWT 인증
- 비밀번호 해싱 (bcrypt)
- 개인정보 보호
