# 실버세대 - 시니어 커뮤니티 앱

실버세대를 위한 모바일 커뮤니티 플랫폼입니다.

## 📱 주요 기능

### 1. 모임
- 반경 1.5km 내 모임 검색
- 지도로 모임 위치 확인
- 카테고리별 모임 (운동, 문화, 식사, 카페, 노래, 취미, 산책, 독서)
- 모임 참여 및 관리

### 2. 커뮤니티
- 게시글 작성 및 공유
- 댓글 기능
- 좋아요/싫어요 반응
- 정보 공유 및 소통

### 3. 공동구매
- 함께 구매하면 저렴한 상품
- 실시간 참여 인원 확인
- 마감 시간 카운트다운
- 카테고리별 상품 (과일, 정육, 곡물, 수산 등)

### 4. 취미활동
- 서예, 골프, 수필, 수채화, 요가, 사진 등
- 강사 정보 및 일정
- 난이도별 프로그램
- 수강 신청

### 5. 동네병원
- 내 주변 병원 검색
- 병원 예약하기
- 픽업 요청 서비스
- 진료 과목별 검색

## 🚀 실행 방법

### 모바일 앱 실행

```bash
cd mobile
npm install
npm start
```

### 테스트 방법

#### 옵션 1: 실제 기기 (권장)
1. Expo Go 앱 설치 (App Store / Play Store)
2. `npm start` 실행
3. QR 코드 스캔
4. 모든 기능 정상 작동

#### 옵션 2: 웹 브라우저
```bash
cd mobile
npm start --web
```
- http://localhost:19006 에서 확인
- 대부분의 기능 확인 가능

#### 옵션 3: iOS 시뮬레이터 (Xcode 필요)
```bash
cd mobile
npm run ios
```

#### 옵션 4: Android 에뮬레이터
```bash
cd mobile
npm run android
```

## 📦 기술 스택

- **React Native 0.81.5**
- **Expo SDK 54**
- **React Navigation** - 네비게이션
- **React Native Maps** - 지도 기능
- **Expo Location** - GPS 위치
- **Zustand** - 상태 관리
- **Axios** - HTTP 클라이언트
- **TypeScript** - 타입 안정성
